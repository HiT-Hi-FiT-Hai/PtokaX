TmrMan metatable

TmrMan.AddTimer(nTimerInterval)- Add new timer for script. nTimerInterval is in ms. Return nil when failed or uTimerId when success.
TmrMan.AddTimer(nTimerInterval, sFunctionName)- Add new timer for script. nTimerInterval is in ms. sFunctionName is name of function called by this timer. Return nil when failed or uTimerId when success.
TmrMan.RemoveTimer(uTimerId)- Remove timer with given ID from script.
